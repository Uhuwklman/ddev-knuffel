Overview & Usage

Knuffel is a Python-based console and web application inspired by the classic dice game. You can interact with the app via the web interface (when enabled) or directly in the terminal.

  • Web Interface: Use the Flask app for a graphical experience (if enabled).
  • Terminal: Use the browser terminal to play the console version.
  • Monitoring: Check system metrics and status.

Basic Usage:
- For the terminal version, log in or register, then follow the on-screen instructions.
- For the web version, use the navigation links and forms provided.
- Note: Database changes in the demo may not persist.
- Commands available when logged into SSH:
checkupdate - runbackup - updateknuffel
